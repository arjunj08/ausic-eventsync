## 🎯 What Each Part Does

### Frontend (Mobile App)
- **LoginScreen** - User authentication with role selection
- **AdminDashboard** - Event & team management console
- **CreateEventScreen** - Create new events
- **TeamManagementScreen** - Configure teams & members
- **EventDetailMemberView** - View published events
- **TeamRoomScreen** - Team updates & operations
- **MyTasksScreen** - Personal task management
- **CrossTeamRequestScreen** - Inter-team collaboration
- **NotificationsScreen** - Alert center
- **MemberProfileScreen** - User profile
- **TeamChatScreen** - Group messaging
- **DirectMessageScreen** - DM conversations
- **VoiceCallScreen** - Voice call interface
- **VideoCallScreen** - Video call interface
- **CallsChatsHubScreen** - Communication hub

### Backend (REST API)
- **Auth API** - User registration, login, token management
- **Events API** - Create, read, update, delete events
- **Teams API** - Manage teams and members
- **Tasks API** - Task creation and tracking
- **Cross-Team API** - Collaboration requests with approvals
- **Messages API** - Chat and DM functionality
- **Updates API** - Team announcements
